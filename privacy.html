Push Pal — Privacy Policy

Last updated: June 27, 2026

Push Pal is a personal fitness coaching application used solely by its owner (Justin Chung). It is not a commercial product and is not distributed to third parties.

Data Collected

Push Pal connects to WHOOP's API to read the following data belonging solely to the app owner:

  • Recovery scores, HRV, and resting heart rate
  • Sleep duration and sleep stage data
  • Daily strain and cycle data
  • Workout data

How Data Is Used

All data retrieved from WHOOP is stored locally in a SQLite database on the owner's personal computer. It is used exclusively to generate personalized training recommendations via an AI coaching assistant. No data is transmitted to any third party other than Anthropic's API (for AI coaching responses), and no data is stored on any remote server.

Data Sharing

No personal data is shared with any third party. The app has no user accounts, no backend infrastructure, and no analytics. Data never leaves the owner's local machine except for API calls to WHOOP (to read data) and Anthropic (to generate coaching responses).

Data Retention

Data is retained in a local SQLite database on the owner's personal computer for as long as the owner chooses to keep it. The owner may delete the database file at any time to remove all stored data.
